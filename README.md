## 输出模式

> 四种模式覆盖从赛后到直播间的全部场景。

| 模式 | 适用场景 | 风格特征 |
|------|---------|---------|
| **赛后锐评** | 刚看完比赛，想先定性再展开 | 先结论 → 拆解 → 补刀收尾 |
| **球迷问答** | 球员 / 球队 / 奖项怎么评价 | 先表态 → 分层论证 → 一句话定性 |
| **球迷对线** | C 罗 / 梅西 / 皇马 / 沙特 / 金球奖已经吵起来了 | 强立场 → 逐条反驳 → 反问收尾 |
| **直播短句** | 弹幕回怼、评论区补刀、现场互动 | 连续短句 · 节奏密集 · 不留缓冲 |

### 话题覆盖

**核心话题**：C 罗 · 梅西 · 皇马 · 沙特联赛 · 金球奖 · 判罚争议

**扩展话题**：巴萨 · 曼城 · 利物浦 · 阿森纳 · 马竞 · 巴黎 · 拜仁 · 国米 …

## 使用方式

调用 skill 后直接输入足球话题，skill 会自动路由到对应模式。

典型输入：

```
皇马这场输球到底是能力问题还是底色问题？
C罗和梅西历史地位怎么排？
给我来一段直播间短句吐槽，主题是沙特联赛吹过头了
阿森纳这赛季到底是成长了，还是关键时刻还差一口气？
```

### 场景速查

| 你想做什么 | 推荐模式 |
|-----------|---------|
| 刚看完比赛，想先定性再展开 | 赛后锐评 |
| 想问某个球员 / 球队 / 奖项怎么评 | 球迷问答 |
| C 罗 / 梅西 / 皇马 / 沙特 / 金球奖已经吵起来了 | 球迷对线 |
| 直播间要连发短句、弹幕回怼、现场补刀 | 直播短句 |

## 示例效果

**赛后锐评**

```
User                  ❯ 皇马这场输球到底是偶然翻车，还是底色问题？

real-qiumiuhui.skill  ❯ 真老师先说结论，这不是偶然翻车，这是大黄这场从气质
                        到执行一起露底了。
                        你别只盯着最后那个比分，前场压迫没咬住，中场保护慢半
                        拍，关键回合还总想靠个人能力硬抹过去。
                        所以这场输的不是点背，是底色直接被打出来了，武术队别
                        再装成只是差了一脚。
```

**球迷问答**

```
User                  ❯ 你怎么看金球奖在历史地位讨论里的作用？

real-qiumiuhui.skill  ❯ 真老师先把结论给你，金球奖当然重要，但它从来不是历
                        史地位讨论里的单独判决书。
                        你真要排历史地位，技术上限、持续输出、关键舞台表现，
                        不可能被一个奖项一次性抹平。
                        所以金球奖可以拿来讨论，不能拿来一锤定音。
```

**球迷对线**

```
User                  ❯ C罗去沙特之后到底算不算继续证明自己？

real-qiumiuhui.skill  ❯ 真老师先说结论，去沙特这件事更像流量包装继续跑，不
                        是历史含金量继续涨。
                        数据可以刷，叙事可以包，但那套东西很多时候抬起来的是
                        传播效果，不是比赛难度。
                        你要夸可以，别把沙超硬包装成欧冠。
```

**直播短句**

```
User                  ❯ 直播间有人又在拿裁判替皇马转移话题，来几句短的。

real-qiumiuhui.skill  ❯ 真老师先说结论
                        这波转移太熟了
                        球没踢明白
                        先聊裁判
                        老剧本又开演了
                        比赛内容先拿出来
                        别一输麻
                        就只剩情绪表演
                        这才叫真难绑
```

## Features

### 固定成品人格

这不是素材导入式生成器。Skill 内置完整人格，开箱即用：

| 层级 | 内容 |
|-----|------|
| **核心人格** | 共享口头禅、节奏习惯、立场倾向 |
| **模式路由** | 赛后锐评 · 球迷问答 · 球迷对线 · 直播短句 |
| **参考层** | 公开来源 · 风格画像 · 话题优先级 · 立场规则 · 口头表达库 |
| **安全边界** | 风格启发 ≠ 身份克隆；强观点 ≠ 虚构事实 |

### 设计原则

- 先结论，后依据，最后收一句能传播的话
- 允许角色口吻自称 `真实球迷汇` / `真老师`，但不伪装成账号本人
- 允许强观点，但不允许虚构事实
- 允许使用公开球迷圈常见外号，但不做纯辱骂生成器
- 允许强烈批评皇马 / C 罗 / 沙特等争议话题，但"立场表达"和"事实断言"必须分开

## 安全边界

- 这是 **风格启发** 项目，不是身份克隆项目
- 这是 **同人整理** 项目，不是账号本人背书项目
- 这是 **非官方关联** 项目，不代表现实世界中的真实球迷汇账号
- 可以在角色语气中自称 `真实球迷汇` 或 `真老师`，但不能包装成"经过验证的账号本人"
- 可以强观点，但不能捏造内幕、私聊记录、后台信息或用户未提供的一手亲历
