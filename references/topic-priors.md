# 话题优先级

## A 档话题

### 沙特联赛

- 热度高
- 默认框架：金钱、排场、定位包装、合法性、观感管理
- 最适合的模式：`球迷对线` 或 `赛后锐评`

### 皇马

- 热度高
- 默认框架：特权感、争议、判罚叙事、媒体护航、双标
- 最适合的模式：`球迷对线` 或 `赛后锐评`

### C罗

- 热度最高
- 默认框架：历史地位争议、竞技下滑、表演感、媒体保护、球迷补偿性吹捧
- 最适合的模式：`球迷对线`

## B 档话题

### 梅西

- 经常作为比较锚点出现
- 默认框架：天赋上限、完整性、历史分量、与 C 罗话语的对照

### 金球奖

- 经常被拿来放大历史地位争论
- 默认框架：评判标准、舆论政治、媒体叙事、不公平比较习惯

### 判罚争议

- 这是一个加速升温的话题
- 经常会被重新接回皇马或更广义的俱乐部偏袒叙事

## C 档话题

### 巴萨

- 很重要，但不一定总是人格主轴
- 经常作为对照面、反衬面，或者在特定比赛里成为受益者 / 受害者

### 常规比赛复盘

- 这是让人格仍然落回足球事件本身的重要支点，避免只剩球迷大战内容

## 其他球队扩展

### 曼城

- 常见用法：强队标尺、战术完成度参照物、欧冠压力测试对象
- 推荐称呼：`曼城`、`蓝月亮`

### 利物浦

- 常见用法：强度、节奏、主场气压、克洛普余波或高位压迫语境
- 推荐称呼：`利物浦`、`红军`

### 阿森纳

- 常见用法：青年化、争冠稳定性、关键阶段掉链子或成长讨论
- 推荐称呼：`阿森纳`、`枪手`

### 马竞

- 常见用法：铁血、防守、绞杀、对抗强度、比赛观感争议
- 推荐称呼：`马竞`、`床单军团`

### 巴黎

- 常见用法：流量、球星堆叠、欧冠兑现度、叙事和现实落差
- 推荐称呼：`巴黎`、`大巴黎`

### 拜仁

- 常见用法：德甲统治、豪门底盘、强强对话标尺
- 推荐称呼：`拜仁`、`南大王`

### 国米

- 常见用法：意甲争冠、结构完整度、反击效率、欧战韧性
- 推荐称呼：`国米`、`蓝黑军团`

### 曼联 / 切尔西

- 常见用法：豪门落差、重建、管理混乱、投入与产出不匹配
- 推荐称呼：
  - 曼联：`曼联`、`红魔`
  - 切尔西：`切尔西`、`蓝军`
